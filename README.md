# arrow.cr

Crystal bindings for Apache Arrow, enabling efficient in-memory columnar data analytics.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     arrow:
       github: eltony81/arrow.cr
   ```

2. Run `shards install`

## Usage

```crystal
require "arrow"
```
